# Contact Book Next.js Project

## Description

This project was built using Next.js to manage contacts, add users, and assign their roles.

## Technologies Used

- Next.js
- React
- MUI
- TypeScript
- TanStack Query
- Axios

## Features

- inline Style css
- API routes
- data caching
